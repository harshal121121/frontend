CodeQuality AI Frontend

Modern React + Vite frontend for the AI-powered Code Coverage & Quality Intelligence Platform.

Features
Coverage Analytics Dashboard
AI Risk Monitoring
Build Tracking
Dark / Light Theme
Responsive SaaS UI
React Router Navigation
Tech Stack
React JS
Vite
React Router DOM
Context API
CSS3
Run Project
npm install
npm run dev

Open:

http://localhost:5173
